📘 JPA Mess Management (Supplier-Item)

📂 Package

com.mess.main

📌 Description

This Java program uses JPA (Hibernate) to manage relationships between Supplier and Item entities through a Supply mapping table.

It demonstrates:
•	Data insertion using EntityManager
•	JPQL queries for fetching data

⸻

🚀 Features
•	Insert Supplier, Item, and Supply records
•	Fetch all supplier names
•	Fetch suppliers from a specific city (Delhi)
•	Fetch items supplied by a particular supplier

⸻

▶️ Queries Used
•	Get all supplier names
•	Filter suppliers by city
•	Join query between Supplier and Item
