📌 Description

This Java program demonstrates various operations on arrays including:
•	Printing arrays (for loop & for-each)
•	Sum, minimum, maximum, and second maximum
•	Average calculation
•	Operations on even/odd indices
•	Splitting array into first half and second half

🚀 Features
•	Array traversal techniques
•	Basic mathematical operations
•	Index-based calculations
•	Half-array operations
