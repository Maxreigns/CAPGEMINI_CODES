📘 Custom UserMap with Exception Handling

📂 Package

src.org.example

📌 Description

This program implements a custom UserMap by extending HashMap to prevent duplicate usernames.
•	If a username already exists, a custom exception is thrown.
•	Demonstrates method overriding and custom exception handling in Java.

🚀 Features
•	Custom HashMap implementation
•	Duplicate key restriction
•	Custom runtime exception (UsernameExistsException)
•	Clean validation logic
