💡 Description

This week focuses on database interaction using Java. It starts with JDBC for direct database communication and progresses to Hibernate, an ORM (Object Relational Mapping) framework that simplifies database operations.

🚀 Key Learnings
•	Connecting Java applications to databases
•	Performing CRUD operations using JDBC
•	Understanding ORM concepts
•	Mapping Java objects to database tables using Hibernate
•	Structuring applications with DAO, Model, and Repository layers
